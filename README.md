Calculator App (Flutter)

A simple and fully functional Calculator App built using Flutter.
It performs all basic arithmetic operations with a clean and responsive user interface.

# Features

Perform basic arithmetic: Add, Subtract, Multiply, Divide

AC (All Clear) and Backspace  functionality

Dark / Light theme toggle with saved preference

Prevents invalid inputs (like multiple operators in a row)

Built with clean and modern UI using Flutter
